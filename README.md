# ac-multihack-external
Assault cube external multihack

![изображение](https://github.com/user-attachments/assets/f669bb40-25fc-4c80-b167-3ffbe6842837)


# Features
Visuals:
  - 2D boxes
  - 3D boxes
  - Snaplines
  - Health
  - View direction
  - Color pickers

Aim:
  - Vector Aim
  - Magic bullets
  - Auto gun
    
Misc:
  - No recoil
  - No knockback
  - No firerate
  - Infinite ammo
  - No grenade delay
  - Fly hack

# How-to-use
  Start assault cube and open multihack, to open menu press insert

# Build
  release x86
